Common Utilities
================

Utility classes for general use, such as:
* Base 64 encoding
* Command execution
* Sending emails
* Handle passwords
* File directory management
* String utilities
* XML utilities
* Basic cryptography
* ZIP input/output

Note: This code was taken from the [RODA](https://github.com/keeps/roda) project, so it can be used in other open-source projects.
